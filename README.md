# Gemini Nano Chat (Unofficial)

Chat with Gemini Nano — a local AI running directly in your browser. This is an unofficial extension and is not affiliated with or endorsed by Google. No data is sent to any server.

## Features

- **Local AI:** Powered by the built-in Chrome Prompt API (Gemini Nano) for on-device processing.
- **Privacy First:** Your chat stays entirely on your device. Zero data is transmitted to cloud servers.
- **Side Panel Interface:** Conveniently chat with the AI from the browser's side panel without disrupting your workflow.
- **No API Keys:** Fully utilizes the browser's built-in AI capabilities, so no external API keys or subscriptions are required.

## Requirements

To use Gemini Nano Chat, you must meet the hardware and browser requirements for the built-in AI:

- **Browser**: Chrome version 138 or higher.
- **Hardware**: At least 16 GB of RAM, 4 CPU cores (for CPU inference) or a GPU with strictly more than 4 GB VRAM (for GPU inference with audio multimodal input support), and 22 GB of free disk space to download the local model.

For full details, see the official [Chrome Built-in AI Documentation](https://developer.chrome.com/docs/ai/prompt-api).

## Installation

1. Download or clone this repository.
2. Go to `chrome://extensions` in your Chrome browser.
3. Enable "Developer mode" (top right).
4. Click "Load unpacked" and select the `extension` folder.

**Also available on the [Chrome Web Store](https://chromewebstore.google.com/detail/gemini-nano-chat-unoffici/gmakmkbabhpggomefcdggdncdbpbajpo)**
_(Note: The Chrome Web Store version may not always have the latest updates.)_

## Usage

- Click the extension icon to open the Gemini Nano Chat in your browser's side panel.
- Type your prompts and chat directly with the local AI.
